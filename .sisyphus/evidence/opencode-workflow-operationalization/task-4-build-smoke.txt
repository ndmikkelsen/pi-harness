Usage: ai-harness [options] [command] [project] [target]

AI workflow scaffolder for local setup of new and existing projects.

Arguments:
  project                    project name or target path
  target                     target directory

Options:
  --assistant <assistant>    assistant target: codex or opencode (default:
                             "codex")
  --mode <mode>              scaffold mode: auto, new, existing (default:
                             "auto")
  --init-json                emit machine-readable JSON output for scaffold runs
                             (default: false)
  --dry-run                  show planned changes without writing files
                             (default: false)
  --force                    overwrite managed files (default: false)
  --merge-root-files         in existing-project mode, merge scaffold entries
                             into .gitignore and .env.example (default: false)
  --cleanup-manifest <id>    in existing-project mode, apply a curated cleanup
                             manifest before scaffolding
  --non-interactive          disable prompts and report prompt-required cleanup
                             actions instead (default: false)
  --skip-git                 skip git initialization (default: false)
  --detect-ports             probe the compute host for available service ports
                             (default: false)
  --dolt-port <port>         explicit Dolt port
  --cognee-db-port <port>    explicit Cognee database port
  --compute-host <host>      compute host for generated configs (default:
                             "10.10.20.138")
  --compute-user <user>      SSH user for generated configs (default: "compute")
  --ssh-key-path <path>      SSH private key path for generated configs
                             (default: "~/.ssh/z3r0Layer-main")
  -h, --help                 display help for command

Commands:
  doctor [options] [target]  Audit whether a repository is scaffolded correctly
                             for Codex/OpenCode.
  install-skill [options]    Install the global OpenCode skill bundle for
                             ai-harness.
