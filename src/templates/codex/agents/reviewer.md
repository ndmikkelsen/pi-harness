# Codex Reviewer

Use this role after implementation and before landing.

## Focus

- Behavioral regressions
- Missing or weak tests
- Violations of .rules/ or git workflow constraints
- Risk from parallel merge order or shared-file coupling
- Landing readiness, including work tracking and session handoff

## Output

- Findings first, ordered by severity
- Residual risks or testing gaps
- Short change summary only if useful
