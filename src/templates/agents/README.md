# .agents

Role briefs for reusable AI workflow agents used across scaffold sessions.

- `scaffold-developer.md` — implement scoped work with minimal, safe edits.
- `gsd-cognee-advisor.md` — gather concise context before planning and execution.
- `reviewer.md` — validate completed work before handoff.

Guidance:

- Choose one role per work packet and follow its operating mode.
- Keep outputs concise: responsibilities, rules, operating mode, and explicit risks.
- Reuse these templates across projects; avoid stack-specific instructions.
- When a tool is unavailable, continue with local repo context.
