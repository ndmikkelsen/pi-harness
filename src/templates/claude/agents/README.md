Reusable role briefs for this scaffolded workspace.

- `scaffold-developer.md` — implement scoped work with safe, minimal changes.
- `gsd-cognee-advisor.md` — gather context and define next steps before execution.
- `reviewer.md` — run final validation before handoff.

Guidance:

- Keep role outputs short and actionable.
- Prefer file-level evidence and explicit risks.
- Stay within the selected role's scope; avoid cross-scope redesign.
