# v1.0 Milestone

## Goal

Ship the first complete `ai-harness` milestone: a preserve-by-default scaffold engine for new and existing repositories, one shared Codex/OpenCode runtime surface, a globally installed `harness` skill, and explicit local-use distribution and migration guidance.

## Scope

- Phase 1 - scaffold engine and shared runtime foundation (`CORE-01`, `CORE-02`, `CORE-03`)
- Phase 2 - self-hosted harness rollout and global skill flow (`CORE-04`, `CORE-05`)
- Phase 3 - local-use distribution, upgrade guidance, and migration hardening (`CORE-06`)

## Completion Standard

- All v1 core flows and quality gates in `.planning/REQUIREMENTS.md` are marked done.
- Planning artifacts agree that the active milestone is complete.
- Verification evidence exists for the later hardening phases and no blocker follow-up issues remain open for v1 scope.

## Artifacts

- `SUMMARY.md` - milestone outcome, deliverables, and follow-on work
- `AUDIT.md` - final audit of v1 requirement coverage and release readiness
