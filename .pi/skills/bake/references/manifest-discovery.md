- `pnpm-lock.yaml`
- `pnpm-workspace.yaml`
- `turbo.json`
- `tsconfig.json`

## Python
- `pyproject.toml`
- `requirements.txt`
- `poetry.lock`

## Go
- `go.mod`

## Rust
- `Cargo.toml`

## Java / Kotlin
- `pom.xml`
- `build.gradle`
- `build.gradle.kts`

## Containers and deploy
- `Dockerfile*`
- `docker-compose*.yml`
- `compose*.yml`
- `config/deploy*.yml`
- `.kamal/`
